# HTML
## 一、标签
### <a href="./source/label/基础.md">1.基础</a>
| 标签         | 描述               |
|--------------|--------------------|
| `<!DOCTYPE>`   | 定义文档类型。     |
| `<html>`       | 定义一个 HTML 文档 |
| `<title>`      | 为文档定义一个标题 |
| `<body>`       | 定义文档的主体     |
| `<h1> to <h6>` | 定义 HTML 标题     |
| `<p>`          | 定义一个段落       |
| `<br>`         | 定义简单的折行。   |
| `<hr>`         | 定义水平线。       |
| `<!--...-->`   | 定义一个注释       |
