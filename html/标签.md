# 一、第一个html
```
<!DOCTYPE html> 
<html lang="en">
 <head> 
 <meta charset="UTF-8"/> 
 <title>我的第一个页面</title>
 </head>
 <body>
 hello world! 
 </body>
</html
```
# 二、开发工具
## 2.1 文档声明标签
`<!DOCTYPE html>`
当前页面采取的是 HTML5 版本来显示网页<br>
 <!DOCTYPE>不是一个 HTML 标签，它就是 文档类型声明标签。<br>
## 2.2 lang 语言种类
用来来定义当前文档显示的语言。
1. en定义语言为英语
2. zh-CN定义语言为中文

## 2.3 字符集
在<head>标签内，可以通过<meta> 标签的 charset 属性来规定 HTML 文档应该使用哪种字符编码。
` <meta charset=" UTF-8" />`
